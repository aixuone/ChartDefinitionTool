/charts

/charts post

post

保存表单定义

post /charts

保存表单定义

  • Request
  • Response

Query Parameters

  • chartName: required (string)

    表单名称

  • chartTheme: required (string)

    表单主题

  • chartType: required (string)

    表单类型

  • dataSetID: required (string)

    数据集ID

  • columnValue: required (array of object)

    值字段

    Items: columnValue

    • columnID: required (string)

      字段ID

    • columnDispType: required (string)

      展现方式

    • columnPolymer: (one of SUM, COUNT)

      聚合类型

  • columnOption: (array of object)

    条件字段

    Items: columnOption

    • columnID: required (string)

      字段ID

    • columnType: (one of DATE, TIMESTAMP, TEXT - default: TEXT)

      查询类型

  • columnRelation: (array of object)

    穿透字段

    Items: columnRelation

    • columnID: required (string)

      字段ID

    • relatedColumnUUID: required (string)

      聚合字段UID

  • columnAxis: required (array of object)

    轴字段

    Items: columnAxis

    • columnID: required (string)

      字段ID

    • columnPolymer: (one of GROUP, YEAR, SEASON, MONTH, WEEK - default: GROUP)

      聚合方式

  • columnSeries: (array of object)

    系列字段

    Items: columnSeries

    • columnID: required (string)

      系列字段ID

HTTP status code 200

Body

Media type: application/json

Type: object

Properties
  • success: required (one of success, error)

    成功标志

  • message: required (string)

    返回信息

  • status: required (string)

    返回状态

  • data: required (string)

    表单ID
